**PEWER TyPes** is a premium, feature-rich speed typing application inspired by classic typing tools like Monkeytype. Built with modern React, TypeScript, and high-performance vanilla CSS, it offers a visual and auditory typing experience complete with mechanical switch sounds, a real-time keyboard visualizer, and custom vector-based telemetry charts.

## ✨ Features

- 🎹 **Mechanical Keyboard Sound Synthesizer**
  - Experience the tactile satisfaction of mechanical switches. Includes simulated key sounds for **Cherry MX Blue (Clicky)**, **Cherry MX Brown (Tactile)**, and **Cherry MX Red (Linear)** switches, complete with master volume control.
- 🎨 **Dynamic Theme Engine**
  - Instant theme switching without reloading. Includes beautifully curated palettes like Carbon, Dracula, Cyberpunk, Forest, Sakura, and retro Terminal.
- ⌨️ **Live Keyboard Visualizer**
  - Watch your keystrokes fire in real-time on a beautifully responsive virtual on-screen keyboard.
- 📈 **Real-Time SVG Telemetry & Metrics**
  - Interactive SVG line chart graphing your WPM (Words Per Minute) and raw speed second-by-second. Shows mistake tracking (red cross-marks) exactly where they occurred.
- ⚙️ **Typing Configurations & Carets**
  - **Multiple Modes**: Choose between *Time* limits (15s, 30s, 60s, 120s), *Word count* targets (10, 25, 50, 100), or preset *Quotes* (short, medium, long).
  - **Custom Practice**: Input your own text snippets to drill specific articles or code blocks.
  - **Caret Styles**: Customize the caret as a line, block, underline, or hide it.
- 🚀 **Interactive Particles & Physics**
  - Typing speed directly drives visual background particle speeds, creating an interactive typing environment.

## 🎹 Switch Sound Settings

Tailor your sound feedback to match your mechanical keyboard style:

- **MX Blue (Clicky)**: High-pitched tactile click feedback.
- **MX Brown (Tactile)**: Subtle, soft mechanical click feedback.
- **MX Red (Linear)**: Smooth, deeper bottom-out sound profile.
- **Mute**: Focus mode with silent inputs.

## 🚀 Getting Started

Follow these steps to run **PEWER TyPes** locally:

### Prerequisites

- Node.js (v18.0 or higher recommended)
- npm (v9.0 or higher)

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/pewer-types.git
   cd "PEWER Types"
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start the Dev Server**
   ```bash
   npm run dev
   ```

4. **Open in Browser**
   Navigate to the local dev URL printed in your terminal (typically `http://localhost:5173`).

## 🛠️ Tech Stack & Architecture

- **Frontend**: [React 19](https://react.dev/) using functional hooks for state synchronization.
- **Languages**: [TypeScript](https://www.typescriptlang.org/) for robust types across states.
- **Bundler**: [Vite](https://vite.dev/) for extremely fast Hot Module Replacement (HMR).
- **Icons**: [Lucide React](https://lucide.dev/) for clean vector interfaces.
- **Styling**: Modern, fluid variables and animations in pure Vanilla CSS.
